# 串口

串口资产通过串口控制台连接设备 —— 适用于网络设备、嵌入式开发板和带外管理。

## 创建串口资产

创建一个串口资产（见[资产管理](/docs/guide/asset-management)）：

- **端口路径** —— 串口设备路径（必填），例如 Windows 上的 `COM3` 或 Linux/macOS 上的 `/dev/ttyUSB0`
- **波特率** —— 连接速率（必填），例如 `9600` 或 `115200`
- **数据位** —— `5`、`6`、`7` 或 `8`（默认：`8`）
- **停止位** —— `1`、`1.5` 或 `2`（默认：`1`）
- **校验位** —— `none`、`odd`、`even`、`mark` 或 `space`（默认：`none`）
- **流控** —— `none` 或 `hardware`

## 使用控制台

打开该资产以将终端连接到串口。会话使用所配置的线路参数呈现设备的串口控制台，并提供与其他 OpsKat 终端相同的终端体验（见 [SSH 终端](/docs/guide/ssh-terminal)）。
